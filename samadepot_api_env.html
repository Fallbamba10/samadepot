API Routes et variables d'environnement SamaDepot

API Routes
Variables d'environnement
Authentification
Gestion de la connexion et des sessions
MéthodeRouteDescriptionAccès
POST/api/auth/loginConnexion email + mot de passe — détecte l'université via le domainePublic
POST/api/auth/logoutDéconnexion — invalide la session JWTAuth
GET/api/auth/meRetourne l'utilisateur connecté (profil + université + rôle)Auth
POST/api/auth/reset-passwordDemande de réinitialisation du mot de passe par emailPublic
GET/api/auth/[...nextauth]Handler NextAuth.js — gère OAuth callbacks et sessionPublic
Espaces de dépôt
Créés par les professeurs — consultés par les étudiants
MéthodeRouteDescriptionAccès
GET/api/spacesListe les espaces actifs de l'université de l'utilisateur connectéStudent / Teacher
POST/api/spacesCrée un nouvel espace de dépôt (titre, type, deadline, formats, taille max)Teacher
GET/api/spaces/[id]Détail d'un espace (infos + liste des dépôts reçus)Teacher / Admin
PATCH/api/spaces/[id]Modifier un espace (deadline, titre, activation/désactivation)Teacher
DELETE/api/spaces/[id]Supprimer un espace (uniquement si aucun dépôt reçu)Teacher
GET/api/spaces/[id]/exportExporter la liste des dépôts en CSV (nom, statut, date, note)Teacher / Admin
Dépôts (Submissions)
Table centrale — upload + suivi des fichiers
MéthodeRouteDescriptionAccès
GET/api/submissionsMes dépôts (étudiant) ou tous les dépôts de mes espaces (prof)Auth
POST/api/submissionsCréer un dépôt — valide format/taille, génère hash SHA-256, sauvegarde en BDDStudent
GET/api/submissions/[id]Détail d'un dépôt (fichier, statut, review si existante)Auth
DELETE/api/submissions/[id]Supprimer son dépôt (seulement si statut = received)Student
GET/api/submissions/[id]/receiptGénérer et télécharger le récépissé PDF (horodatage + hash)Student
GET/api/submissions/[id]/downloadGénérer une URL signée temporaire (15 min) pour télécharger le fichierTeacher / Admin
Upload fichiers
Endpoint dédié — validation + stockage Supabase Storage
MéthodeRouteDescriptionAccès
POST/api/uploadUpload un fichier — valide MIME type, taille, génère hash SHA-256, stocke dans Supabase Storage bucket privéStudent
POST/api/upload/presignedGénère une URL présignée pour upload direct côté client (fichiers > 10 Mo)Student
Évaluations (Reviews)
Retour du professeur sur chaque dépôt
MéthodeRouteDescriptionAccès
POST/api/submissions/[id]/reviewSoumettre une évaluation (décision + note optionnelle + commentaire)Teacher
PATCH/api/submissions/[id]/reviewModifier une évaluation (seulement dans les 24h)Teacher
Notifications

MéthodeRouteDescriptionAccès

Authentification
NEXTAUTH_SECRETClé secrète JWT — générer avec : openssl rand -base64 32Requis
NEXTAUTH_URLURL du site — ex: https://samadepot.sn (ou http://localhost:3000 en dev)Requis
Supabase — Base de données & stockage
NEXT_PUBLIC_SUPABASE_URLURL du projet Supabase — ex: https://xxxx.supabase.coRequis
NEXT_PUBLIC_SUPABASE_ANON_KEYClé publique (anon) — lecture seule soumise au RLSRequis
SUPABASE_SERVICE_ROLE_KEYClé admin — bypass RLS — JAMAIS exposée côté clientRequis
SUPABASE_STORAGE_BUCKETNom du bucket privé pour les fichiers — ex: submissionsRequis
SUPABASE_STORAGE_BUCKET_RECEIPTSBucket pour les récépissés PDF — ex: receiptsRequis
WhatsApp Business API (Meta)
WHATSAPP_PHONE_IDID du numéro WhatsApp Business — depuis Meta for DevelopersRequis prod
WHATSAPP_TOKENToken d'accès permanent WhatsApp Business APIRequis prod
WHATSAPP_VERIFY_TOKENToken de vérification webhook — valeur aléatoire de ton choixRequis prod
